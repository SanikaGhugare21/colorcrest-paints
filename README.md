# ColorCrest Paints — Landing Page

A responsive, visually appealing landing page for the fictional paint brand **ColorCrest Paints**, showcasing product categories with clear CTAs.

## Live Demo
Add your Netlify/Vercel deployment link here: `https://your-live-link.example`

## Tech Stack
- HTML5
- CSS3 (no framework; pure CSS with Grid/Flex)
- Vanilla JavaScript (basic interactivity + form validation)
- Responsive: Desktop, Tablet, Mobile

## Sections
- **Hero** — Brand name, tagline, banner feel with color swatches, primary CTA (“Explore Colors”)
- **Product Categories** — 4 cards (Interior, Exterior, Wood, Metal) with placeholder images and “Learn More” CTAs
- **Why Choose Us** — 4 benefit highlights (eco‑friendly, quick drying, long‑lasting, many shades)
- **Contact / Inquiry** — Simple form with client‑side validation and success message
- **Footer** — Brand, quick links, placeholder social icons

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in your browser — no build step required.

## Deploy (Netlify)
1. Push to GitHub.
2. On Netlify, choose **Add new site → Import from Git** and pick your repo.
3. Leave build settings empty (static site). Click **Deploy**.
4. Copy the live URL and paste it above in **Live Demo**.

## Deploy (Vercel)
1. Push to GitHub.
2. On Vercel, **Add New… → Project**, import your repo.
3. Framework preset: **Other** (static).
4. Deploy and copy the live URL.
